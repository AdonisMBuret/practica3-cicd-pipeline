# Práctica 3: Pipeline CI/CD

## Descripción
App web simple con pipeline CI/CD usando GitHub Actions.

## Instalación
1. Clona el repo: `git clone URL`
2. `npm install`
3. `npm test`

## Pipeline
- Build: Instala deps y corre pruebas.
- Deploy: A GitHub Pages en push a main.

## Pruebas
- Unitarias: En tests/app.test.js
- Cobertura: `npm run test:coverage`

URL de la app desplegada: https://tu-usuario.github.io/practica3-cicd-pipeline/
